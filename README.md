- Signup/Login
- Create team
- Join team using code/leave team
- Easy to use.
- Displays all details about the event.

## Instructions to run
```
$ git clone https://github.com/jaynikunjshah/stockastic23-frontend/
$ cd stockastic23-frontend
$ npm i
$ npm run dev
```
